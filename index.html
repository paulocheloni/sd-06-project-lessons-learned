PRIMEIRO BLOCO

Um pouquinho do bloco Introdução ao desenvolvimento web

  1. Internet: Compreender os conceitos básicos de rede e os fundamentos da internet.
  2. Unix & Bash: Utilizar, com familiaridade, os comandos básicos da linha de comando,executando programas, navegando e manipulando arquivos.
  3. Git: Utilizar a ferramenta, seguindo um modelo padronizado de branching e merging.
  4. HTML: Aplicar a linguagem de forma semântica, utilizando, com familiaridade, seus elementos.
  5. CSS: Utilizar de forma correta e organizada os seletores da linguagem para aplicar estilos em marcações HTML.
  6. Javascript: Introduzir os conceitos da programação estruturada. Utilizar de forma eficiente os recursos básicos do Javascript.
  7. HTML:Utilizar a linguagem de marcação endereçando a responsividade e a acessibilidade, utilizando tags semânticas, rich contents e otimização para SEO.
  8. CSS + uma vez: Aplicar layouts responsivos, reutilizando estilos, com alta coesão e baixo acoplamento à marcação.
  9. Javascript: Utilizar as principais features do ES6, criar testes unitários, desenvolver código assíncrono utilizando callbacks e Promises.
  10. React: Desenvolver componentes React testáveis, manipulando estado nos componentes e globalmente usando ferramentas como Redux, ContextAPI e React Hooks, e bibliotecas RTL para testar componentes síncronos e assíncronos.

Muita coisa, né? Agora, vejam as cenas para os próximos capítulos:

  • Banco de dados:Modelar entidades de dados e seus relacionamentos e propriedades de forma relacional e não relacional.
  • REST: Compreender os conceitos básicos de uma API REST, Utilizando os verbos HTTP na construção e consumo de uma API.
  • Server-side: Implementar de forma eficiente uma API REST,passando por todo o ciclo de vida de uma requisição HTTP no back end.
  • Autenticação: Compreender a diferença entre autenticação e autorização, e as diferentes formas de aplicar ambos os conceitos.
  • Segurança: Aplicar as melhores práticas de segurança no desenvolvimento de aplicações web.
  • Deploy: Publicar uma web app em um ambiente de produção, utilizando boas Práticas de versionamento e segurança.
  • Algorítmos: Aplicar técnicas de resolução de problemas com algoritmos de forma eficiente, além de reconhecer e otimizar níveis de complexidade.
  • Estrutura de dados: Compreender as diferenças entre as principais estruturas de dados, e como empregá las da melhor forma de acordo com o problema dado.
  • Paradigmas de programação:Compreender os principais conceitos e motivações dos paradigmas de orientação a objetos e funcional.
  • Python: Solidificar o conhecimento de lógica de programação, apresentando conceitos como gerenciamento e compartilhamento de memória.
