Day 3

Neste dia aprendi sobre Bash, uma forma de interagir com Unix por meio do terminal, utilizando comandos! Aprendi também sobre as estruturas de diretórios do sistema e sobre os comandos que podem ser executados no terminal.

Day 4

Continuamos a explorar os comandos do Unix, neste dia aprendemos alguns dos principais comandos,operadores e conceitos de permissões de arquivos e diretórios.

Lista de comandos que aprendi:

  1. man
  2. ls
  3. cd
  4. cp
  5. mv
  6. cat
  7. grep
  8. rm
  9. find
  10. clear

Day 5

Aprendi sobre Shell Script, um passo além de utilizar os comandos do Unix para criar scripts utilizando o Shell. Criamos utilizamos scripts utilizamos a linguagem Shell Script e os comandos do Linux para automatizar tarefas diárias. Aprendemos também sobre como utilizar condicionais, laços de repetição e a ler dados do usuário usando Shell Script
